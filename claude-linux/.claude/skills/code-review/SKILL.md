# Code Review Skill

コードレビューを体系的に実施するためのスキルです。

## レビュー観点

### 1. 機能性

- 要件を満たしているか
- エッジケースの考慮
- エラーハンドリング

### 2. 可読性

- 命名規則の一貫性
- コードの構造と整理
- コメントの適切さ

### 3. セキュリティ

- 入力バリデーション
- 認証・認可
- 機密情報の取り扱い

### 4. パフォーマンス

- 不要な処理の有無
- N+1問題
- メモリ使用量

### 5. テスト

- テストカバレッジ
- テストの品質
- エッジケースのテスト

## 出力形式

```markdown
## Code Review Report

### Summary
[全体的な評価と主要な発見事項]

### Good Points
- [良い点1]
- [良い点2]

### Issues Found

#### [Critical/High/Medium/Low]: [問題タイトル]
- **File**: [ファイルパス:行番号]
- **Issue**: [問題の説明]
- **Suggestion**: [修正提案]

### Recommendations
[改善のための提案]
```
