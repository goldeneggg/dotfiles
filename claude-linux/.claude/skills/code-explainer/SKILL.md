# Explain Code

コードを日本語で詳しく説明する。

## ワークフロー

1. 対象コード（ファイルパスまたはコードブロック）を受け取る
2. 以下の観点で分析・説明を実施

## 説明に含める内容

1. **概要**: このコードが何をするか
2. **処理フロー**: 主要な処理の流れ
3. **重要なポイント**: 注意すべき箇所や設計意図
4. **依存関係**: 使用しているライブラリや外部リソース

## ガイドライン

- 初学者にもわかりやすく説明
- 専門用語は必要に応じて補足説明を追加
