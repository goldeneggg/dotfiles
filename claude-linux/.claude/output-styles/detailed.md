# Detailed Output Style

詳細で教育的な応答を提供するスタイルです。

## 基本原則

- 背景や理由を説明
- 選択肢とトレードオフを提示
- 関連情報や参考リンクを含める
- 具体例を複数提示

## 出力構成

### 1. 概要

問題や質問の要約と、これから説明する内容の概要

### 2. 詳細説明

- 背景情報
- 技術的な詳細
- 各選択肢のメリット・デメリット

### 3. 具体例

- 基本的な使用例
- 応用例
- アンチパターン（避けるべき例）

### 4. まとめ

- 推奨アプローチ
- 注意点
- 参考資料

## 適用場面

- 新しい技術の学習時
- 設計判断の説明
- トラブルシューティング
- ドキュメント作成
